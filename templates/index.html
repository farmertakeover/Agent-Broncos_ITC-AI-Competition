What it does

  • Multi-turn chat with tool-based corpus search (FAISS + embeddings)
  • Grounded answers with source attribution
  • Optional voice input and read-aloud on the chat page
  • Topic map of crawled filenames (not graph RAG)

How retrieval works

This project uses vector RAG (semantic search over chunk embeddings stored in FAISS), not graph RAG. The corpus map is a visual overview of page topics from filenames only.
